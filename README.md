## Projects

### 1. Competitor Monitoring Agent
**Tools:** n8n · Gemini · Cheerio · HTTP nodes · JavaScript  
**Domain:** E-commerce competitive intelligence (Wayfair)

A fully automated competitive intelligence pipeline that scrapes live product data from Wayfair, Amazon, and Walmart simultaneously, feeds it into a sequential Gemini LLM agent system, and outputs three structured business reports automatically.

**What it does:**
- Ingests competitor product URLs via chat trigger
- Classifies URLs into products vs. collections, then routes each to parallel scraping pipelines
- Extracts pricing, ratings, product features, and size options using Cheerio and custom JavaScript parsing nodes
- Merges data from 5 concurrent pipelines into a single structured dataset
- Passes merged data to a Gemini agent with a detailed system prompt covering 3 intelligence missions:
  - **Feature Gap Report** — identifies missing product features vs. competitors with specific remediation actions and revenue impact estimates
  - **Pricing Action Report** — calculates exact price differentials, quality-adjusted gaps, and recommends specific pricing moves with financial projections
  - **Content Strategy Report** — compares copy tone, keyword strategy, and photo approaches, then recommends exact copy changes with expected conversion lift
- Post-processes AI output to clean HTML and exports as a downloadable report

**Architecture highlights:**
- Parallel scraping with error continuation pipeline doesn't break if one source fails
- Custom URL classifier handles both direct product links and category collection pages
- Retry logic with 5-second backoff on the LLM agent node
- Output rendered as formatted HTML with download option

### 2. Wayfair Content & Trend Agent
**Tools:** n8n · Gemini · Cheerio · Walmart API · Amazon API · JavaScript  
**Domain:** E-commerce content automation and trend analysis

An end-to-end content generation pipeline triggered by product URLs. Scrapes best-selling product data from Amazon and Walmart, generates market trend signals using a first LLM agent, then passes those signals to a second LLM agent that produces editorial-quality blog content and Instagram captions.

**What it does:**
- Accepts Amazon and Walmart product and collection URLs via chat input
- Classifies and routes URLs in parallel across 4 scraping paths (Amazon products, Amazon collections, Walmart products, Walmart collections)
- Extracts product names, prices, sale data, size variants, color options, and product descriptions
- Combines all scraped data into a unified dataset
- Agent 1 (Trend Editor persona): Analyzes product data and generates trend signals (e.g., "Neutral earth tones up 25% in Q4 search volume")
- Agent 2 (Brand Copywriter persona): Converts trend signals into editorial blog posts, "What's In vs. What's Out" tables, and high-engagement Instagram captions
- Cleans and exports final output as formatted HTML

**Architecture highlights:**
- Dual sequential agent pipeline with distinct system prompts and personas
- 5-input merge node coordinating all parallel scraping branches
- Data normalization layer that handles both array and object input structures
- Product categorization logic (rug vs. mat vs. carpet) built into the combine node
- Retry logic on both agent nodes with 5-second wait between attempts
